2.环形loading动效原理：6个小圆点，以圆心外的一个点(transform-orgin设置)旋转，通过animation-delay设置延迟，出现延续效果。

    .dot{
        width: 5px;
        height: 5px;
        background: #fff;
        border-radius: 100%;
        position: absolute;
        top: 50%;
        left: 40%;
        -webkit-box-shadow: 0 0 3px white;
        box-shadow: 0 0 3px white;
        transform-origin:calc(2.5px + 30px) 2.5px;
        opacity: 0;
        animation:rotate 2s ease infinite;
    }
    @keyframes rotate{
        0%{opacity: 1;transform:rotate(90deg);}
        50%{width: 5px;height: 5px;opacity: 0.5;}
        100%{transform:rotate(450deg); opacity: 1;}
    }
    .dot:nth-child(2){animation-delay:.05s;}
